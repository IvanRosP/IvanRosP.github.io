Información Técnica de la Página

Lenguajes o tecnologías utilizadas

  • HTML: Utilizado para la estructura y organización del contenido de la página web.
  • SCSS: Empleado como preprocesador de CSS para mejorar la organización de estilos, reutilización de código y mantenimiento del diseño.
  • Less: Utilizado como apoyo en la gestión de estilos, facilitando la creación de hojas de estilo más dinámicas y estructuradas.

Plataforma de publicación

El sitio web se encuentra publicado mediante la plataforma GitHub, utilizando GitHub Pages como servicio de alojamiento. Esta plataforma permite la publicación de contenido estático de forma segura, accesible y confiable, facilitando además la actualización continua del sitio.

Control de versiones

Para el control de versiones se utiliza un repositorio en GitHub que funciona como respaldo y sistema de seguimiento de cambios. Dicho repositorio permite almacenar versiones históricas del proyecto, recuperar información en caso de errores y mantener un respaldo constante del contenido, asegurando la integridad y continuidad del desarrollo de la página web.
